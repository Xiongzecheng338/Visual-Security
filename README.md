### 📖 项目简介

本项目不仅仅是一个展示画廊，更是一次**认知科学的探索之旅**。它打破了传统静态展示的局限，通过动态交互让你亲手"触摸"错觉，并从神经科学、数学与物理学角度揭开感知的奥秘。

**核心亮点：**
*   🎨 **视觉盛宴**：赛博朋克风格 UI，玻璃态设计，沉浸式体验。
*   🧠 **硬核科普**：包含数学公式推导、神经科学原理及物理学解释。
*   🖱️ **高度交互**：点击图形触发变化，亲身体验"眼见不为实"。
*   🚀 **极致轻量**：单文件架构，无需安装环境，双击即可运行。
